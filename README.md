# Online vs Offline Learning Performance Analysis

## Project Overview
This project compares student performance in online and offline learning environments to understand which method leads to better academic outcomes based on exam scores, focus level, retention, and study hours.

---

## Dataset Access
To view the dataset in its raw form, download:
**Online_vs_offline_learning_database DONE.xlsx**

---

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Basic Statistical Functions  
- Data Visualization  

---

## Dataset Overview
The dataset includes student information across different subjects and learning modes with the following variables:
- Study Hours  
- Focus Level  
- Retention Score  
- Exam Score  
- Subject  
- Learning Mode  

---

## Methodology
- Data cleaning and preparation  
- Exploratory Data Analysis using pivot tables  
- Comparison of averages across learning modes  
- Correlation analysis between key variables  
- Visual dashboard creation  

---

## Key Insights
- Offline learners showed slightly higher average exam performance than online learners  
- Retention and focus levels were also slightly higher in offline learning  
- Study hours had very weak correlation with focus and retention, suggesting other factors influence performance  

---

## Subject Performance Insights
- English and History performed better in online learning  
- Mathematics, Programming, and Science performed better in offline learning  
- Learning effectiveness varies depending on subject area  

---

## Conclusion
Offline learning shows slightly better overall performance in this dataset, but subject differences suggest that no single learning mode is universally better for all disciplines.
